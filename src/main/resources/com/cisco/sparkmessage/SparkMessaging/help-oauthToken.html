An OAuth2 Token for a valid Spark user in the given Spark Room is required to add a Machine Account to a Spark Room. Delete after the account is added.

To obtain an OAuth2 Token:

  1. Go to web.ciscospark.com and sign in
  2. Right click anywhere and select "Inspect Element"
  3. Select the "Resources" tab
  4. Then select "Cookies"->"web.ciscospark.com"
  5. The OAuth token is to the right of "Authorization" in the form "Bearer OAUTH_TOKEN"
